# 📝 React Task Manager Application

## 📌 Project Overview
The React Task Manager is a simple productivity application built using React.js.  
It allows users to:

- Add tasks
- Delete tasks
- Mark tasks as completed
- Persist tasks using Local Storage
- View dynamic UI updates in real-time

This project demonstrates core React concepts such as components, hooks, state management, and side effects.

---

## 🎯 Objectives
- Understand React component-based architecture
- Use React Hooks (useState, useEffect)
- Implement CRUD operations
- Manage state efficiently
- Persist data using Local Storage
- Build reusable UI components

---

## 🛠️ Tech Stack
- React.js (Functional Components)
- JavaScript (ES6+)
- HTML5
- CSS3
- Local Storage API
